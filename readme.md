# Main purpose:
To search most popular words in text file.
## Usage:
./trivial [path to file] [name of text (optional)]
## Example:
./trivial file.txt Call of Cthulhu
## Output:
```
Top-10 most popular words in Call of Chtulhu
1. и: 421
2. в: 323
3. на: 149
4. не: 142
5. что: 122
6. с: 112
7. из: 91
8. он: 85
9. я: 83
10. его: 83
```
